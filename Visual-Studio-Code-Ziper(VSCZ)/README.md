# Visual Studio Code Ziper (VSCZ)

VS Codeのエクスプローラーでフォルダーやファイルを右クリックするだけでZIP圧縮・解凍ができる拡張機能です。

## 機能

- エクスプローラーのコンテキストメニューから「ZIP」で圧縮
- ZIPファイルを右クリックして「Unzip」で解凍
- フォルダー内のディレクトリ構造を完全に保持（空フォルダー含む）
- 圧縮済みファイル（.jpg, .png, .mp4等）は無圧縮格納でサイズ増加を防止
- ZIP作成後の整合性検証
- Zip Slip攻撃に対するセキュリティ保護
- 進捗表示付き処理（圧縮・解凍両方）

## 使い方

### ZIP作成
1. VS Codeのエクスプローラーでフォルダーまたはファイルを右クリック
2. 「ZIP」を選択
3. 進捗通知が表示され、ZIPファイルが作成されます

### ZIP解凍
1. ZIPファイルを右クリック
2. 「Unzip」を選択
3. 同じディレクトリに解凍されます（同名フォルダーがある場合は上書き確認）

## インストール

### installer.cmd を使う場合
1. visual-studio-code-ziper-0.0.2.vsix が同じフォルダーにあることを確認
2. installer.cmd をダブルクリック
3. VS Codeの `code` コマンドが PATH に通っている必要があります

### 手動インストール
1. VS Codeを開く
2. Ctrl+Shift+P でコマンドパレットを開く
3. 「Extensions: Install from VSIX...」を選択
4. .vsix ファイルを指定

## 要件

- VS Code 1.108.1 以上
