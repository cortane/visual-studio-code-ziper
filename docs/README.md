# Visual Studio Code Ziper

> Right-click. Compress. Done.

---

VS Codeのエクスプローラー上で右クリックするだけで ZIP 圧縮・解凍ができる拡張機能です。  
コマンドライン操作は一切不要。直感的な操作で ZIP ファイルの作成・展開が可能です。

---

## [ Features ]

```
  +-- ZIP圧縮 ----------------------------------------------------+
  |                                                              |
  |  * 右クリックメニューから「ZIP」を選ぶだけで圧縮             |
  |  * フォルダー構造を完全保持（空フォルダー含む）          |
  |  * 圧縮済みファイル(.jpg .png .mp4等)は自動で無圧縮格納  |
  |    -> サイズ増加を防止しつつ、最適な圧縮率を実現       |
  |  * 作成後に自動で ZIP 整合性検証を実行             |
  |                                                              |
  +--------------------------------------------------------------+

  +-- ZIP解凍 ----------------------------------------------------+
  |                                                              |
  |  * ZIP ファイルを右クリック -> 「Unzip」で即座に展開        |
  |  * エントリごとの進捗表示付き                           |
  |  * Zip Slip 攻撃に対するセキュリティ保護             |
  |  * 同名フォルダーがある場合は上書き確認ダイアログ        |
  |                                                              |
  +--------------------------------------------------------------+
```

---

## [ Usage ]

### -- ZIP 作成 --

```
  1. エクスプローラーで対象を右クリック
  2. コンテキストメニューから「ZIP」を選択
  3. 進捗通知が表示され、完了後に成功メッセージが表示
```

| 入力 | 出力 |
|------|------|
| `myfolder/` | `myfolder.zip` |
| `document.txt` | `document.zip` (\* 解凍時: `document/document.txt`) |

### -- ZIP 解凍 --

```
  1. ZIP ファイルを右クリック
  2. 「Unzip」を選択
  3. 同じディレクトリに展開されます
```

| 入力 | 出力 |
|------|------|
| `archive.zip` | `archive/` |

## [ Smart Compression ]

本拡張機能はファイルごとに最適な圧縮方式を自動選択します。

```
  テキストファイル (.txt .json .xml ...)  -->  DEFLATE (最大圧縮)
  画像ファイル   (.jpg .png .webp ...)  -->  STORE   (無圧縮)
  動画ファイル   (.mp4 .mkv .avi ...)   -->  STORE   (無圧縮)
  音声ファイル   (.mp3 .flac .aac ...)  -->  STORE   (無圧縮)
  アーカイブ     (.zip .gz .7z ...)      -->  STORE   (無圧縮)
```

既に圧縮されているファイルを DEFLATE すると逆にサイズが増加します。  
STORE を使うことで、Windows 標準 ZIP と同等以上の圧縮効率を実現します。

## [ Security ]

```
  [x] Zip Slip 攻撃保護  --  展開先パスの事前検証
  [x] ZIP 整合性検証   --  作成後の自動ベリファイ
  [x] エラー時クリーンアップ  --  失敗時に壊れた ZIP を自動削除
  [x] 上書き防止     --  既存ファイルへの意図しない上書きをブロック
```
